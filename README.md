# IoT Based E-Vehicle Charging Station

## Project Description
This project is an IoT-based Electric Vehicle Charging Station that allows users to check charger availability and book charging slots through a mobile application. The system uses WiFi communication to send real-time data to a server.

## Objectives
- Reduce waiting time at EV charging stations
- Enable online booking of charging slots
- Monitor charging status remotely

## Technologies Used
- Arduino / ESP8266
- WiFi (IoT)
- Mobile Application
- Cloud Server
- Embedded C

## Working Principle
The charging station is connected to the internet using WiFi. User requests from the mobile app are sent to the server, which communicates with the charging unit to control and monitor the charging process.
